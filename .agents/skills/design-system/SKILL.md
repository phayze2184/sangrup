# Design System Rules

## Typography
- H1 → Space Grotesque
- Body → Inter
- Eyebrow → DM Mono

## Colors
- Use CSS variables only
- No hardcoded hex values in components

## Spacing
- Use consistent spacing scale (8px system)

## UI Principles
- Clean, technical, industrial feel
- Avoid visual noise
- Prioritize readability and hierarchy

## When to use
- Any UI decision
- Styling components
