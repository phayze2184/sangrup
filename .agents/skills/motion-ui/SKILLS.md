# Motion UI Guidelines

## Principles
- Subtle, not distracting
- Support UX, not decoration

## Animation
- Use easing: cubic-bezier(0.22, 1, 0.36, 1)
- Duration: 200ms–400ms

## Performance
- Use transform + opacity only
- Avoid layout-triggering properties

## Tools
- Prefer Motion One (JS)
- Avoid heavy libraries

## When to use
- Hero animations
- Micro-interactions
